# Описание API

## 1. Что это за API

Restful-Booker — это учебный API для практики тестирования CRUD-операций и авторизации.

## 2. Какие сущности важны

Ключевые сущности проекта:

- auth token;
- booking;
- booking id;
- пользовательский доступ к защищённым операциям.

## 3. Какие операции будут интересовать QA

С точки зрения QA наиболее важны:

- получение токена;
- получение списка бронирований;
- получение конкретного бронирования;
- создание нового бронирования;
- полное обновление бронирования;
- частичное обновление бронирования;
- удаление бронирования.

## 4. Почему этот API подходит для портфолио

Этот API подходит для портфолио, потому что позволяет показать:

- работу с HTTP-методами;
- проверку статусов ответа;
- проверку request / response body;
- работу с авторизацией;
- негативные API-сценарии;
- проектирование API test cases;
- использование Postman как рабочего инструмента QA.
